# WyckoffWaveM5EA 参数说明

## 策略定位

`WyckoffWaveM5EA.mq5` 是一套面向 `M5` 的多结构交易 EA，核心把市场分成三类可交易结构：

- `Wave3`：识别 `1-2` 结构完成后，做第 `3` 浪启动
- `Wave5`：识别 `1-2-3-4` 完成后，做第 `5` 浪延续
- `Range`：识别区间中的 `Spring / Upthrust` 假突破反转

默认参数当前是偏稳健的 `AUTO` 档，适合先在 `XAUUSD M5` 上做回测和模拟观察。

---

## 默认模式说明

当前默认配置的核心思路：

- 优先用 `AUTO` 自动判断当前更像趋势还是震荡
- 趋势环境优先尝试 `Wave3`，其次尝试 `Wave5`
- 震荡环境尝试 `Range Spring / Upthrust`
- 使用更高周期 `M15` 趋势过滤
- 使用较高的最小波动门槛、较低的单笔风险、较少的日内开仓次数

这套默认值偏向：

- 少做单
- 避开噪音行情
- 优先做结构更清晰的机会

---

## 快速建议

- 初次使用，先保持默认值不动，先回测 `XAUUSD M5`
- 如果你觉得单子太少，优先下调 `MinATRPoints`
- 如果你觉得假突破太多，优先上调 `TrendGapATR`、`Wave3MinStructureATR`
- 如果你觉得交易太激进，优先下调 `RiskPercent`、`MaxTradesPerDay`

---

## 参数说明表

### 基础参数

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `TradeTimeframe` | `PERIOD_M5` | 策略执行周期，当前设计就是围绕 `M5` 节奏 | 建议保持 `M5` |
| `MagicNumber` | `20260414` | 本 EA 的订单标识，用来区分其它 EA 或手工单 | 多个 EA 同时跑时要改成不同值 |
| `EnableLong` | `true` | 是否允许做多 | 只做空时可关掉 |
| `EnableShort` | `true` | 是否允许做空 | 只做多时可关掉 |
| `CloseOnOpposite` | `true` | 出现反向信号时是否先平现有单 | 保守风格建议开启 |
| `MaxPositions` | `1` | 最大持仓数，`0` 表示不限制 | 新手建议保持 `1` |
| `MaxSpreadPoints` | `80` | 最大允许点差，超过则不新开仓 | 黄金点差大时可适度放宽到 `100~120` |
| `TradeDeviation` | `20` | 下单允许滑点 | 过小容易拒单，过大容易劣价成交 |

### 多结构模式

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `EntryMode` | `ENTRY_AUTO` | 自动选择 `Wave3 / Wave5 / Range` | 推荐先用 `AUTO` |
| `Wave3MinStructureATR` | `2.2` | `1-2` 结构最小波幅，越大越严格 | 想多做单可降到 `1.8~2.0` |
| `TrendGapATR` | `0.18` | 识别趋势状态时，快慢均线间距至少达到多少个 `ATR` | 趋势过滤更严格可升到 `0.20~0.25` |
| `RangeGapATR` | `0.05` | 识别震荡状态时，快慢均线间距不超过多少个 `ATR` | 震荡识别太宽可降到 `0.03~0.04` |
| `RangeBreakoutATR` | `0.08` | 区间反转信号棒突破确认缓冲 | 假突破多时可升到 `0.10~0.12` |

### 执行过滤

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `UseSessionFilter` | `false` | 是否限制开仓时段 | 当前默认全天允许开仓 |
| `SessionStart` | `06:00` | 允许开仓开始时间 | 仅在启用时段过滤时生效 |
| `SessionEnd` | `23:00` | 允许开仓结束时间 | 仅在启用时段过滤时生效 |
| `MinBarsBetweenEntries` | `8` | 两次开仓最少间隔的 `M5` K 线数 | 想减少频繁交易可升到 `10~12` |
| `MaxTradesPerDay` | `2` | 每日最大开仓次数，`0` 表示不限制 | 稳健模式建议 `1~2` |
| `UseDailyLossLimit` | `true` | 是否启用日内亏损熔断 | 实盘建议开启 |
| `DailyLossLimitCurrency` | `200.0` | 日内累计亏损达到该金额后停止新开仓 | 要按账户大小调整 |

### 仓位参数

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `UseRiskPercent` | `true` | 是否按风险比例自动算仓位 | 推荐开启 |
| `RiskPercent` | `0.5` | 单笔风险占余额百分比 | 新手建议 `0.25~0.5` |
| `FixedLots` | `0.10` | 固定手数模式使用的手数 | 仅在关闭 `UseRiskPercent` 时生效 |

### 波浪参数

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `BarsToScan` | `260` | 每次识别结构时回看的 K 线数量 | 太小会漏结构，太大会更慢 |
| `PivotStrength` | `3` | 分型强度，越大越不敏感 | `2` 更灵敏，`4` 更稳 |
| `MinStructureATR` | `3.0` | `Wave5` 结构最小整体幅度 | 只做大结构可继续提高 |
| `Wave2MinRetrace` | `0.20` | 第 2 浪最小回撤比例 | 太小会接受过浅回调 |
| `Wave2MaxRetrace` | `0.79` | 第 2 浪最大回撤比例 | 太大说明结构容易失真 |
| `Wave3MinRatio` | `1.25` | 第 3 浪至少达到第 1 浪的多少倍 | 越大越强调真正的推动浪 |
| `Wave4MinRetrace` | `0.15` | 第 4 浪最小回撤比例 | 太小容易把横盘噪音当结构 |
| `Wave4MaxRetrace` | `0.62` | 第 4 浪最大回撤比例 | 太大容易把破坏结构的行情放进来 |
| `RequireNoOverlap` | `false` | 是否强制第 4 浪不和第 1 浪重叠 | 更纯粹的推动浪可开成 `true` |
| `BreakoutBufferATR` | `0.10` | 波浪突破确认缓冲 | 假突破多时可加大 |
| `PivotMatchATR` | `0.80` | 威科夫信号与关键 pivot 匹配容差 | 越小越严格 |

### 威科夫参数

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `TradingRangeBars` | `24` | 定义区间边界时参考的 K 线长度 | 区间策略更慢可提高到 `30~40` |
| `WyckoffSignalBars` | `12` | 最近多少根 K 线内查找 `Spring / Upthrust` | 太大容易信号过旧 |
| `VolumeLookback` | `20` | 成交量均值回看周期 | 一般保持 `20` 左右 |
| `FalseBreakATR` | `0.12` | 假突破至少超出区间多少个 `ATR` | 越大越严格 |
| `VolumeMultiplier` | `1.20` | 当前量至少达到均量多少倍才算有效 | 假信号多时可提到 `1.3~1.5` |
| `CloseStrengthBuy` | `0.60` | `spring` 收盘强度阈值 | 越大表示收盘必须更强 |
| `CloseStrengthSell` | `0.40` | `upthrust` 收盘强度阈值 | 越小表示收盘必须更弱 |

### 趋势与出场

| 参数 | 默认值 | 说明 | 调整建议 |
|---|---:|---|---|
| `EMAFastPeriod` | `34` | 当前周期快均线 | 可与慢线一起调，但不建议太短 |
| `EMASlowPeriod` | `89` | 当前周期慢均线 | 与快线一起定义趋势方向 |
| `UseHTFFilter` | `true` | 是否启用更高周期趋势过滤 | 稳健模式建议开启 |
| `HTFTimeframe` | `PERIOD_M15` | 高周期过滤所用周期 | `M15` 很适合 `M5`，也可试 `H1` |
| `HTFEMAFastPeriod` | `34` | 高周期快均线 | 一般与当前周期参数保持一致 |
| `HTFEMASlowPeriod` | `89` | 高周期慢均线 | 一般与当前周期参数保持一致 |
| `ATRPeriod` | `14` | ATR 周期 | 常用标准值 |
| `MinATRPoints` | `140.0` | 最小波动门槛，低于该点数不交易 | 交易太少可降到 `100~120` |
| `StopBufferATR` | `0.25` | 结构高低点外的止损缓冲 | 假扫损多时可加大 |
| `RewardRisk` | `2.00` | 固定止盈风险收益比 | 稳健模式建议 `1.8~2.2` |
| `EnableBreakEven` | `true` | 是否启用保本止损 | 推荐开启 |
| `BreakEvenAtRR` | `1.00` | 浮盈达到多少倍风险时推保本 | 更激进可降到 `0.8` |
| `EnableTrailing` | `true` | 是否启用移动止损 | 推荐开启 |
| `TrailATRMultiplier` | `1.20` | 移动止损距离 ATR 倍数 | 太小容易被甩出，太大又会回吐 |

## `EntryMode` 使用建议

### `ENTRY_AUTO`

适合大多数情况，先判断趋势还是震荡，再自动调用相应结构。

适合：

- 还在摸索参数阶段
- 希望 EA 自己判断市场环境
- 希望减少人工切换模式

### `ENTRY_WAVE3_ONLY`

只做 `3` 浪启动，通常利润空间最大，但信号会更早、更敏感。

适合：

- 更想抓趋势主段
- 能接受一定的假突破增加

### `ENTRY_WAVE5_ONLY`

只做 `5` 浪延续，更偏确认后入场，通常更慢，但有时更稳。

适合：

- 偏保守
- 不想做太早的结构判断

### `ENTRY_RANGE_ONLY`

只做区间 `Spring / Upthrust`，适合偏震荡环境。

适合：

- 确认市场是箱体震荡
- 不想追趋势突破

### `ENTRY_WAVE3_AND_5`

只做趋势结构，不做区间反转。

适合：

- 明确只想做趋势单
- 不想参与震荡洗盘交易

---

## `AUTO` 稳健版适合什么情况

当前默认参数比较适合：

- `XAUUSD M5`
- 想先稳一点、少做一点
- 账户还在测试期
- 更重视回撤控制，而不是极限做单频率

如果后续你要细分，我建议再做三套预设：

- `AUTO 稳健版`
- `Wave3 激进版`
- `Range 震荡版`

这样在不同市场阶段切换会更方便。

---

## 使用顺序建议

1. 先保持默认值，回测最近 `3~6` 个月
2. 先看交易次数、胜率、盈亏比、最大回撤
3. 如果交易太少，先调 `MinATRPoints`
4. 如果假信号太多，再调 `TrendGapATR`、`RangeBreakoutATR`、`VolumeMultiplier`
5. 最后再调 `RiskPercent`

先调过滤，再调仓位，通常比一开始就调手数更有效。
